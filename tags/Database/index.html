一些关于粘土参数的数据库

Ching Jianye1 and Kok-Kwang Phoon2

CPTU-Su (Ching and Phoon, 2012a) -- Establishment of generic transformations for geotechnical design parameters3: PDF-Original, Official Website

摘要：岩土工程设计参数通常基于现场调查结果的转换来估算。通常，人们希望转换不确定性根据数据库中站点的数量和类型而变化。本研究试图解决与转换不确定性有关的以下两个问题：（a）转换不确定性如何随数据库中站点的数量和类型而变化；以及（b）当从一个足够大的数据库开发一个“通用的”转换，转换不确定性是否会最终落在狭窄的范围内。这项研究还试图提出一个框架，以建立这种通用的转换并量化其不确定性。静力触探CPTU数据与粘土的不排水抗剪强度（\(S_u\)）之间的转换证明了该框架。已经发现，CPTU-\(S_u\)变换及其不确定性取决于站点或区域，并且就平均趋势而言，一个站点的“本地”变换方程可能不适用于另一站点（众所周知）以及变异系数（COV）。提出了一种开发通用CPTU-\(S_u\)转换方程的方法，该方程可用于在缺乏本地数据的情况下进行下游可靠性分析或设计。敏感性分析表明，它需要至少15个站点的数据，并附带一个隐含的假设，即充分的地理覆盖范围通常意味着足够的地质多样性，才能可靠地构建此类“通用”转换方程式。

关键词：转换不确定性，岩土设计参数，基于可靠性的设计，相关性，CPTU试验，不排水抗剪强度。

CLAY/5/345 (Ching and Phoon, 2012b) -- Modeling parameters of structured clays as a multivariate normal distribution4: PDF-Original, Official Website

摘要：这项研究探索了将流动性指数，不排水抗剪强度，重塑不排水抗剪强度，预固结应力和结构性粘土（敏感或快粘土）的垂直有效应力建模为多元正态分布的可能性。文献中充斥着两个土体参数之间的相关方程。尽管通过这种方法具有明显的实际实用性，但通过构造多元概率分布函数来一致地合成两个以上土体参数的情况很少见。这项研究汇编了一个庞大的结构化粘土数据库，以构建上述五个土体参数之间的多元概率分布。然后，使用该多元分布来模拟感兴趣的土体参数之间的相关性，并导出用于贝叶斯推断的有用方程式。通过另一个独立的结构性粘土数据库以及文献中提出的经验方程，可以进一步验证这种构造的多元分布和方程。

关键词：相关性，多元正态分布，场地特征，基于可靠性的设计。

CLAY/6/535 (Ching, Phoon and Chen, 2014a) -- Modeling piezocone cone penetration (CPTU) parameters of clays as a multivariate normal distribution5: PDF-Original, Official Website

摘要：这项研究研究了将静力触探（CPTU）锥尖阻力，锥孔后超静孔隙水压力，不排水的剪切强度以及轻度超固结粘土的超固结比建模为多元正态分布的可能性。这是一项持续研究的一部分，该研究旨在开发可用于模拟粘土站点上常见土体参数的多元分布。这项研究建立了一个包含535个数据点的大型数据库，其中同时密切测量了CPTU参数，不排水抗剪强度和超固结比。然后，将多元正态分布用于捕获目标土体参数之间的相关性，并得出用于贝叶斯推断的有用方程。通过另一个由594个数据点组成的独立数据库以及文献中提出的经验方程，可以进一步验证这种构造的多元正态分布和方程。这项研究最有用的结果是提供一种系统的分析方法，以在存在CPTU参数的情况下更新规范化的不排水抗剪强度和超固结比的分布。

关键词：相关性，多元正态分布，场地特征，静力触探（CPTU）试验，基于可靠性的设计。

CLAY/7/6310 (Ching and Phoon, 2013) -- Multivariate distribution for undrained shear strengths under various test procedures6: PDF-Original, Official Website

摘要：粘土的不排水剪切强度（\(S_u\)）不是常数。 特别是，通过不同的试验程序评估的粘土的\(S_u\)值是不同的，因为这些试验可能具有不同的应力状态，应力历史，采样扰动程度和应变率。 这项研究基于大型粘土数据库，通过各种试验程序构建了\(S_u\)的多元概率分布。 该多变量分布基于以下四个因素提供了归一化不排水剪切强度的估计值：试验模式，超固结比，应变率和可塑性。 一旦建立了这种多元分布，就可以使用简单的贝叶斯分析系统地回答有趣的问题，例如，给定一个试验程序的\(S_u\)值，另一试验程序的\(S_u\)值的更新平均值和变异系数是多少？

关键词：不排水抗剪强度，不确定性，相关性，多元分布，现场调查。

CLAY/10/7490 (Ching and Phoon, 2014b) -- Transformations and correlations among some clay parameters the global database7: PDF-Original, PDF-Zh-CN, Official Website

摘要：本研究汇编了来自251个研究的10个粘土参数（标记为 CLAY/10/7490 ）的大型数据库，涵盖了来自全球30个地区或国家的粘土数据。因此，该“全球”数据库所涵盖的数据范围比文献中现有转换模型的标定所依据的范围要广。这些转换模型将试验测量值（例如，锥尖阻力）与适当的设计参数（例如，不排水的剪切强度）相关联。数据库显示的10个黏土参数之间的相关行为与文献中现有的转换模型所显示的相关行为一致。这些转换模型相对于全局数据库的偏差和转换不确定性已得到校准。结果发现，更新的转换模型偏差较小，并且转换不确定性通常相当大。通过合并次级输入参数（例如塑性指数或灵敏度），可以进一步降低如此大的变换的不确定性。在同作者撰写的伴随论文中，从 CLAY/10/7490 构建了耦合这些黏土参数的10维多元概率分布，并提出了一个有用的程序，其中涉及从三个独立的测量值更新两个设计参数的整个双变量概率分布。

关键词：黏土特性，相关性，转换模型，数据库，统计数据。

岩土变异性

Phoon and Kulhawy (1999)8认为岩土变异性的三个主要来源为固有变异性（偶然性），测量误差以及转换不确定性（关于平均相关趋势分析的数据散布）。

这种将岩土变异性归类为偶然性与认知性在某种程度上是建模者的选择9。

Phoon and Kulhawy (1999)10提出一些土体特性的变异系数（COV）准则，用于基于可靠性的设计（RBD）的岩土工程分析。

目前工程人员针对岩土工程参数的变异性的处理一般通过进行多次试验，根据实验结果评估土体特性属性值，一般做法是直接取平均值或在一定置信区间内的最坏情况的值。事实上，现场试验得到的数据本质上是多元的，受到各种因素的影响，为考虑多元参数的影响，Ching and Phoon 1112131415开发了基于贝叶斯机器学习方法的耦合多个黏土参数的多元概率分布模型。

目前的问题

一般来说，岩土工程时试验的结果不直接适用于设计参数，需要用一个转换模型将现场试验的参数与设计参数联系起来。大多数的转换模型都是通过使用回归分析的经验或半经验的数据拟合得到的。而且大多数模型是针对于特定区域而开发的。本研究开发了覆盖全球大量区域的全球数据库，并通过引入辅助输入参数降低模型偏差（即变异系数 COV）从而提升其精确性。

模型研究的黏土参数

本研究涉及到以下10个无量纲参数。

  • 指数性质：
    • 液限（LL）。
    • 塑性指数（PI）。
    • 流动性指数（LI）。
  • 应力和强度：
    • 归一化竖向有效应力（\(\sigma_{\rm v}^\prime/P_{\rm a}\)）。
    • 归一化预固结应力（\(\sigma_{\rm p}^\prime/P_{\rm a}\)）
    • 归一化不排水抗剪强度（\(s_{\rm u}/\sigma_{\rm v}^\prime\)）。
    • 灵敏度（\(S_{\rm t}=s_{\rm u}/s_{\rm u}^{\rm re}\)）。
  • 带孔压的静力触探试验（CPTU）参数：
    • 孔隙压力比（\(B_{\rm q}=(u_2-u_0)/(q_{\rm t}-\sigma_{\rm v})\)）。
    • 归一化锥尖阻力（\((q_{\rm t}-\sigma_{\rm v})/\sigma_{\rm v}^\prime\)）。
    • 归一化有效锥尖阻力（\((q_{\rm t}-u_2)/\sigma_{\rm v}^\prime\)）。

实现程序

上述两表可用于获得感兴趣的粘土设计参数的均值和变异系数（COV）。这对于岩土工程可靠性设计（RBD）至关重要。

  • 选择一个与全球数据库相适应的转换模型，例如 Jamiolkowsky (1985)16提出的 \({\rm OCR}-s_{\rm u}/\sigma_{\rm v}^\prime\) 模型（\({\rm OCR}=2\)）。 \[ s_{\rm u}({\rm mob})/\sigma_{\rm v}^\prime\approx 0.23\cdot{\rm OCR}^{0.8} \]

  • 根据实际目标值与预测目标值之比获得偏差因子（\(b\)）和变异系数（COV），对于 \({\rm OCR}-s_{\rm u}/\sigma_{\rm v}^\prime\) 模型，由表5知，\(b=1.11\)，\({\rm OCR}=0.53\)。则有 \(s_{\rm u}({\rm mob})/\sigma_{\rm v}^\prime\) 的平均值为 \(b\cdot 0.23\cdot{\rm OCR}^{0.8}=1.11\cdot 0.23\cdot 2^{0.8}=0.44\)。

  • 为降低转换不确定性（即降低变异系数 COV），若还给出其它辅助输入参数，如给出 \({\rm PI}=15\) 以及 \(S_{\rm t}=10\) 的信息，则可根据表6据此对均值和变异系数进行修正。\({\rm BCF}=0.71\cdot ({\rm PI}/20)^{0.133}\cdot S_{\rm t}^{0.123}=0.71\cdot ({\rm 15}/20)^{0.133}\cdot 10^{0.123}=0.907\)，\({\rm CCF}=67\%\)。则有 \(s_{\rm u}({\rm mob})/\sigma_{\rm v}^\prime\) 的平均值修正为 \(0.44\cdot 0.907=0.403\)，COV 为 \(0.53\cdot 67\%=0.36\)。

  • 根据其它信息（例如 \({\rm PI}, S_{\rm t}, (q_{\rm t}-\sigma_{\rm v})/\sigma_{\rm v}^\prime, B_{\rm q}\) 等）同样可更新均值和COV。只要建立了这些参数的多元概率分布，就可实现这样的修正17。

CLAY/10/7490 (Ching and Phoon, 2014c) -- Correlations among some clay parameters — the multivariate distribution18: PDF-Original, Official Website

摘要：本文构建了涵盖10个黏土参数的10维多元概率分布。这些参数是液限，塑性指数（PI），流动性指数，有效竖向应力，不排水剪切强度，灵敏度和三个静力触探试验参数。为此，在随附的文件中编译了CLAY/10/7490数据库。该数据库包含来自251个研究的7490个数据点。与每个研究相关的数据点数量从1到419不等，每个研究平均30个数据点。粘土特性涵盖了广泛的超固结比（但大多为1至10），敏感度（\(S_t\)）的广泛范围（\(S_t\) = 1至数十或数百的位置是相当典型的）和广泛的PI（但大部分为8~100）。构造的多元概率分布可以用作先验分布，以基于有限但特定于现场的现场数据得出设计参数的联合分布。请注意，导出的是10个黏土参数的整个联合分布，而不是边际分布，或者仅是均值和变异系数。可以从多个现场测量值更新这些多个设计参数，这比使用当前实践中典型的使用一个现场测量值更新一个设计参数更有用。本文还证明了通过组合可用的双变量模型来构建多元概率模型是可行的，该模型在岩土工程相关文献中很普遍。所提出的方法避免了收集多元数据的需求，而这些数据在典型的现场调查程序中很少见。

关键词：黏土特性，相关性，多元分布，数据库，统计数据。

Frequentist and Bayesian (Ching, Wu and Phoon, 2015) -- Statistical characterization of random field parameters using frequentist and Bayesian approaches19: PDF-Original, Official Website

摘要：由于现场调查收集的信息有限，因此无法获得有关土体属性的平均值、标准偏差和波动规模的实际值。估计值和实际值之间的偏差称为统计不确定性。至少有两种关于如何模拟统计不确定性的学派：频率学派和贝叶斯学派。本文旨在探讨其哲学差异，展示如何根据这两种截然不同的思想体系量化统计不确定性，并比较其表现。为了量化统计的不确定性，置信区间将用于频率学派，而后概率分布将用于贝叶斯学派。将举例说明这两种思想学派在一致性方面的表现。结果表明，一般来说，贝叶斯学派在一致性方面表现更好。特别是，当可用信息量非常有限时，建议使用马尔科夫链蒙特卡罗方法。‎

关键词：统计不确定性，现场调查，频率学派，贝叶斯学派，可靠性。

Bayesian machine learning (Ching and Phoon, 2019) -- Constructing site-specific multivariate probability distribution model using Bayesian machine learning20: PDF-Original, Official Website

摘要：本研究提出了一种新型的数据驱动贝叶斯机器学习方法，用于在岩土工程中构建站点特定的多变量概率分布模型。构建特定于站点的模型有一个权衡：从通用数据开发的模型可能不完全适用于本地站点，但纯粹从有限的站点特定数据开发的模型可能由于重大的统计不确定性而非常不精确。建议的方法基于站点特定数据和通用数据之间的混合，在站点特定数据丰富时由站点特定数据控制，在站点特定数据稀疏时由泛型数据控制。这种方法大致遵循工程师目前根据有限的现场特定信息估算土体参数的方法。该方法承认多变量数据不完整，可以处理现场调查中常见的缺失数据。这是一种贝叶斯方法，因此不确定性被严格量化。实际案例研究用于证明拟议方法的有用性。分析结果表明，该方法可以有效地捕获站点特定数据中的关联行为，而且即使在站点特定数据非常稀疏时，也可以做出有意义的预测。

关键词：岩土工程；场地特征； 多元概率分布模型； 转换模型；不确定性; 贝叶斯机器学习。

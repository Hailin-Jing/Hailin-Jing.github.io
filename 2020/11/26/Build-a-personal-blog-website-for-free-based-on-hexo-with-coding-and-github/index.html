Build a personal blog website for free based on Hexo with Coding and GitHub

This is a blog to build a personal blog website based on hexo with Coding and GitHub. This blog mainly references Docs | Hexo, Getting Started with Icarus.

Installation

There are several things you need to install before building.

Node.js

Node.js provides an official installer for most platforms. For users in mainland China, you can go to Taobao Node.js Mirror to download. There are several Other installation methods:

  • Windows: Install via nvs (recommended) or nvm.

  • Mac: Use Homebrew or MacPorts to install.

  • Linux (DEB/RPM-based) : Install from NodeSource.

  • Others: Use the corresponding package manager to install, you can refer to the Guide provided by Node.js

    For Windows users: When using the official Node.js installer, make sure to check the Add to PATH option (checked by default).

    For Mac/Linux users: If there is an error of EACCES permission when trying to install Hexo, please follow Guidelines issued by npmjs Fix the problem. It is strongly recommended to not use root, sudo, etc. to overwrite permissions. If you installed Node.js using Snap, you may need to manually run npm install in the target folder when initializing a blog.

Open command line, type node --version and npm --version, you will get the version number of Node.js and npm (npm is actually the package manager for Node.js, it was installed when installing Node.js), otherwise the installation failed.

Git

  • For Windows: Click Git - Downloads to download Git and install it.

  • For Mac: Using Homebrew or MacPorts, or download the installer.

  • For Linux (Ubuntu, Debian): sudo apt-get install git-core

  • For Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core

    For Mac users: If you may encounter problems during compilation, please install Xcode in the App Store first. After Xcode is completed, launch and enter Preferences -> Download -> Command Line Tools -> Install to install the command line tools.

    For Windows users: For users in China, you can go to Taobao Git for Windows Mirror to download the git installation package.

Open command line, type git --version, you will get the version number of Git, otherwise the installation failed.

Hexo

After all the necessary applications are installed, you can use npm to install Hexo.

Configure your blog website

You can modify most of the configuration in the _config.yml and _config.icarus file in your local blog path. Once you open the _config.yml file, you may know how to put your personal information in it. For specific configuration, see Configuration | Hexo in English or 配置 | Hexo in Chinese and Icarus User Guide - Configuring the Theme - Icarus in English or Icarus用户指南 - 主题配置 - Icarus in Chinese.

Create a new Coding and GitHub project

Creat a public DevOps Coding project and a public GitHub project named <yourname>.github.io, here yourname is your GitHub name. For Coding, you should open the Pages service manually. You need to go to the project settings page in the homepage of the project, check the 持续部署 check box in the 功能开关 page. Then return to the homepage of the project, Click on the 静态网站 under 持续部署, create a website of 静态网站 type.
